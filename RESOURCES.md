# Resources

## TikZ / PGF

- **PGF/TikZ manual**: `texdoc tikz` — authoritative reference; Part I is a
  gentle tutorial, good starting point.
- **TikZ for the Impatient**: shorter free PDF, good for getting productive
  quickly without reading the full manual.
- **Overleaf TikZ tutorials**: web-based examples you can run immediately.
- **tikz-cd manual**: `texdoc tikz-cd` — separate documentation for
  commutative diagram syntax (quite different from plain TikZ).
